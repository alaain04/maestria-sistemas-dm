Si entrenamos un modelo de machine learning una única vez y nunca lo volvemos a entrenar, nos perderemos la información que estos nuevos datos podrían proporcionarnos para mejorar el modelo. Esto es especialmente importante en entornos donde los comportamientos cambian rápidamente como por ejemplo las compras en linea. En este documento analizaremos cómo manejar los flujos de datos en Python utilizando Apache kafka y también cómo entrenar un modelo de clusterizacion utilizando técnicas de aprendizaje en línea para adaptarse a los flujos de datos entrantes empleando la libreria River.

If we train a machine learning model only once and never train it again, we will miss the information that this new data could provide us to improve the model. This is especially important in environments where behaviors change rapidly, such as online shopping. In this document we will analyze how to handle data flows in Python using Apache kafka and also how to train a clustering model using online learning techniques to adapt to incoming data flows using the River library.
